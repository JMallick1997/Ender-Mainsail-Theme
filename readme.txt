This is an Ender themed set for Mainsail created by JMallick1997. This work is licensed under AGPL v3.0.

IMPORTANT: YOU MUST ENABLE MAINSAIL'S DARK MODE FOR THIS THEME TO WORK PROPERLY!!! 

To install automatically, go to the command line and type:
git clone https://github.com/jmallick1997/MainsailEnderTheme.git ~/printer_data/config/.theme

To update automatically, go to the command line and type:
git -C ~/printer_data/config/.theme pull

To install manually, go to the machine tab, click the gear icon, select 'Show Hidden Files', and then add the .theme file to the 
config file directory list.
